## Overview

Realtime Chat App is a modern chat platform with real-time messaging, authentication, and a responsive UI. Built with React and Firebase for seamless communication.

## Key Features

- **Instant Messaging**: Real-time chat between users.
- **Authentication**: Secure login and protected routes.
- **Responsive UI**: Works on all devices.
- **Firebase Backend**: Uses Firebase Realtime Database for instant updates.

The app provides a fast, secure, and modern chat experience for users.
