## Overview

Hospital Management System is a full-stack application designed to manage hospital operations efficiently. It features patient registration, search, dashboard analytics, and a responsive UI for seamless user experience.

## Key Features

- **Patient Registration**: Add and manage patient records with ease.
- **Search Functionality**: Quickly find patients and records using advanced search.
- **Dashboard Analytics**: Visualize hospital data with charts and statistics.
- **Responsive Design**: Optimized for all devices and screen sizes.

## Technologies Used

## Challenges and Learnings

Building this system provided hands-on experience with database design, RESTful APIs, and data visualization. Implementing responsive design and analytics dashboards improved both technical and UX skills.

## Outcome

The Hospital Management System streamlines hospital workflows, improves data accessibility, and provides actionable insights through analytics.
