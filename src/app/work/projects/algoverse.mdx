## Overview

AlgoVerse is a platform for visualizing and solving pathfinding problems using Dijkstra's algorithm. It features an interactive interface for graph creation and algorithm demonstration.

## Key Features

- **Graph Creation**: Users can create and edit graphs interactively.
- **Dijkstra's Algorithm Visualization**: Step-by-step visualization of the algorithm.
- **Educational Tool**: Ideal for students and educators to learn about graph algorithms.

## Technologies Used

## Challenges and Learnings

Building AlgoVerse improved my understanding of graph theory and algorithm visualization. Creating an intuitive UI for graph editing and algorithm steps was a key challenge.

## Outcome

AlgoVerse helps users understand and experiment with pathfinding algorithms in a hands-on way.
