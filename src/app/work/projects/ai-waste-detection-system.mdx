## Overview

AI Waste Detection System predicts waste type using sensor data and machine learning. It features input validation, prediction confidence, and a user-friendly interface.

## Key Features

- **Sensor Input**: Accepts and validates sensor values.
- **ML Prediction**: Predicts waste type (dry, wet, plastic) using a trained model.
- **Confidence Display**: Shows prediction confidence to users.
- **Web Interface**: Clean, accessible UI for all users.
