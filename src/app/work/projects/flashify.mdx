## Overview

Flashify is an AI-powered app that generates flashcards and quizzes from study materials, tracks learning progress, and works across platforms.

## Key Features

- **Flashcard Generation**: Automatically creates flashcards from uploaded content.
- **Quiz Mode**: Generates quizzes to test knowledge.
- **Progress Tracking**: Monitors learning and adapts to user needs.
- **Cross-Platform**: Works on mobile and web.

## Challenges and Learnings

Integrating AI and cross-platform support was challenging. The project improved my skills in React Native, backend APIs, and AI integration.

## Outcome

Flashify helps users learn efficiently with AI-generated study tools.
